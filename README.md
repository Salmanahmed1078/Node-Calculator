# Node Calculator

A command-line calculator built with Node.js supporting standard arithmetic operations, built as part of a DevOps/SCD lab.

## Features

- Addition, subtraction, multiplication, division
- Modulo and power operations
- Interactive CLI mode
- Input validation and error handling
- Unit tested

## Getting Started

```bash
git clone https://github.com/Salmanahmed1078/Node-Calculator.git
cd Node-Calculator
npm install
node calculator.js
```

## Running Tests

```bash
npm test
```
